Forge Resume
====

## **Introduction**

Forge Resume provides minimilastic and easy approach to create, edit and generate resumes.

## **Demo**

See the live demo at:
https://forgeresume.netlify.app/

## **Installation**

- Download and install Node: https://nodejs.org 
- Clone this repo: git clone https://github.com/yeshasprabhakar/ForgeResume.git (HTTPS)
- Install project dependancies: npm install
- Start the development environment: npm start
- Open your browser and visit http://localhost:3000

## **Features**

- Generate unlimited no of resume
- No registration or login required
- No advertisements or promotions
- Privacy oriented, no collection or sharing of data
- Export as PDF
